# BookEnds
 ## Java Book Management System 
 This is a Java application developed using JPA (Java Persistence API), Spring framework, and Bootstrap for managing books. The application allows users to store and retrieve book information in a MySQL database.  
 
 ### Features: - Add new books with details such as title, author, publication year, and ISBN. 
 - Update existing book information. 
 - Delete books from the database. 
 - Retrieve and display a list of all books. 
 - Search for books based on title, author, or ISBN.
 
 ### Technologies Used: 
 - Java 
 - JPA (Java Persistence API) 
 - Spring Framework 
 - MySQL 
 - Bootstrap
 
 ### Getting Started: To run the application locally, follow these steps:
 1. Clone the repository: `git clone https://github.com/Andessonreis/BookEnds.git` 
 2. Configure the MySQL database connection settings in the `application.properties` file. 
 3. Build the project using a Java IDE or by running `mvn clean install` in the project directory.
 4. Run the application using the IDE or by executing `java -jar target/book-management-system.jar`.
